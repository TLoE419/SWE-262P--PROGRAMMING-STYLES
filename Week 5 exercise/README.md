# Week 5 Exercise - JAR Class Analyzer

## Files

- `JarClasses.java` - Main program for analyzing JAR files
- `json.jar` - Test JAR file (org.json library)

## Compilation

```bash
javac JarClasses.java
```

## Usage

```bash
java JarClasses json-java.jar
```
